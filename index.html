Data Sets

Major Cities

Just over 400 worldwide cities with latitude and longitude coordinates, returned as an array.

<!-- Add the following to your code or download locally -->
      <script src="http://jewkesy.github.io/colloquial/data/majorCities.json" type="text/javascript" />

The data is returned in the following format: -


        var majorCities = [
        { "name": "Argentina, Buenos Aires", "lat": -34.608521, "long": -58.373539},
        { "name": "Argentina, Rio Grande", "lat": -32.0350, "long": -52.0986},
        { "name": "Australia, Adelaide", "lat": -34.926102, "long": 138.599884},
